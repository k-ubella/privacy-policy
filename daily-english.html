영단어 한마디 개인정보처리방침

Privacy Policy · 최종 업데이트: 2026년 3월 15일

1. 개요

영단어 한마디(이하 "앱")는 매일 영어 표현을 학습할 수 있는 앱입니다. 본 개인정보처리방침은 앱을 사용하면서 수집·이용·보관되는 정보에 대해 설명합니다.

본 앱은 개인 식별 정보(이름, 이메일, 연락처 등)를 수집하지 않습니다.

2. 수집하는 정보

기기 내 저장만

앱은 서버로 개인정보를 전송하지 않으며, 다음 데이터를 기기 내에만 저장합니다:

  • 학습 기록: 학습 완료 날짜, 연속 학습일(스트릭), 획득 배지
  • 학습 설정: 알림 설정 시간, 즐겨찾기 문장
  • 캐시 데이터: 문장 목록 원격 캐시 (6시간 TTL)

위 데이터는 앱을 삭제하면 모두 제거됩니다.

3. 광고 (Google AdMob)

본 앱은 Google AdMob을 통해 광고를 제공합니다. Google은 광고 제공을 위해 다음 정보를 수집할 수 있습니다:

  • 광고 ID (GAID / IDFA)
  • 기기 정보 (OS 버전, 기기 모델 등)
  • 앱 사용 패턴 (익명화된 광고 성과 측정)

Google의 개인정보처리방침: https://policies.google.com/privacy

관심 기반 광고 거부: https://adssettings.google.com

4. 알림 권한

앱은 매일 학습 알림을 보내기 위해 알림 권한을 요청합니다. 사용자가 권한을 거부하거나 언제든 설정에서 해제할 수 있으며, 이 경우 알림 기능만 비활성화됩니다.

5. 외부 서비스

앱은 문장 데이터 업데이트를 위해 GitHub Gist에서 JSON 파일을 불러옵니다.

  • 요청 시 전송되는 정보: 없음 (공개 URL GET 요청)
  • 수집하는 개인정보: 없음

6. 아동 개인정보 보호

본 앱은 만 13세 미만 아동으로부터 의도적으로 개인정보를 수집하지 않습니다. 본 앱은 전 연령이 사용할 수 있는 영어 학습 앱입니다.

7. 개인정보처리방침 변경

본 방침이 변경될 경우, 앱 업데이트 릴리즈 노트 및 본 페이지를 통해 공지합니다. 최종 업데이트 날짜는 본 페이지 상단에 표시됩니다.
